# Extract the district assignments for a precinct across all simulated plans

Extract the district assignments for a precinct across all simulated
plans

## Usage

``` r
prec_assignment(prec, .data = pl())
```

## Arguments

- prec:

  the precinct number

- .data:

  a
  [`redist_plans`](http://alarm-redist.org/redist/reference/redist_plans.md)
  object

## Value

integer vector, a row from a plans matrix
