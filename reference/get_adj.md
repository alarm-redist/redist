# Get and set the adjacency graph from a `redist_map` object

Get and set the adjacency graph from a `redist_map` object

## Usage

``` r
get_adj(x)

set_adj(x, adj)
```

## Arguments

- x:

  the `redist_map` object

- adj:

  a new adjacency list.

## Value

a zero-indexed adjacency list (`get_adj`)

the modified `redist_map` object (`set_adj`)
